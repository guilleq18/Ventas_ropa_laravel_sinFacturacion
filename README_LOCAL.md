# Tienda Ropa Laravel

Proyecto Laravel base creado en `E:\Dev\Projects\tienda_ropa_laravel`.

## Stack local instalado

- PHP portable: `E:\Dev\Tools\laravel-runtime\php`
- Composer portable: `E:\Dev\Tools\laravel-runtime\composer\composer.phar`
- Node.js portable: `E:\Dev\Tools\laravel-runtime\node`

## Paquetes agregados

- `laravel/breeze`
- `spatie/laravel-permission`

## Scripts utiles

- `.\scripts\artisan-local.ps1 serve`
- `.\scripts\artisan-local.ps1 migrate`
- `.\scripts\composer-local.ps1 install`
- `.\scripts\composer-local.ps1 dev`
- `.\scripts\npm-local.ps1 run build`

## Base de datos

- `\.env` queda listo con SQLite para arrancar rapido.
- `\.env.mysql.example` deja una base para mover el proyecto a MySQL cuando arranque la migracion real.
